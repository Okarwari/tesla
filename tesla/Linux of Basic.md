#### :a、ta.
他们是一对符号，:a是先做一个标记，然后如果ta之前执行成功，则跳转到:a标识符继续执行，达到了循环的效果。
```
有如下分行的数据：
|      32742 |
|      39013 |
|     481472 |
|     481658 |
|     481885 |
     把格式转换为：32742,39013,481337,481472,481658,481885
     
cat tt | awk '{print $2}' | sed ":a;N;s/\n/,/g;ta"
```
解释: N命令将12、34行用\n拼接在一起，然后循环使用,将\n替换，
