#### JavaScript Function.call() 函数详解
call()函数用于调用当前函数functionObject，并可同时使用指定对象thisObj作为本次执行时functionObject函数内部的this指针引用。

该函数属于Function对象，所有主流浏览器均支持该函数。




### 思考
 * 函数this中带有的属性并不是运行环境时的局部变量。
